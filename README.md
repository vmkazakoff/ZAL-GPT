# Описание проекта

Этот проект представляет собой веб-приложение "MKRobots | Prompt Practice" (также известное как ЗАЛ-GPT), предназначенное для практики и оценки пользовательских промптов для моделей искусственного интеллекта. Пользователи могут получать задания, отправлять свои промпты и получать подробную обратную связь от ИИ, включая оценку по различным критериям.

## Основные компоненты:

*   **Frontend (Клиентская часть):**
    *   Разработан с использованием HTML, Tailwind CSS для стилизации и чистого JavaScript (`scripts/app.js`) для интерактивности.
    *   Использует внешние библиотеки, такие как `marked.js` для рендеринга Markdown, `qrcode.min.js` для генерации QR-кодов и `font-awesome` для иконок.
    *   Обеспечивает пользовательский интерфейс для отображения заданий, ввода промптов, просмотра ответов ИИ и получения детализированной обратной связи.

*   **Backend (Серверная часть):**
    *   Реализован на Google Apps Script.
    *   Обрабатывает запросы на получение заданий (`getPractice`), сохранение информации о пользователях (`saveUser`) и отправку промптов для оценки ИИ (`submitPrompt`).
    *   Взаимодействует с сервисом, подобным OpenAI, для генерации ответов и обратной связи на основе пользовательских промптов.

## Ключевые особенности:

*   **Управление пользователями:** Идентификация пользователей через локальное хранилище и возможность передачи ID через URL.
*   **Задания для практики:** Загрузка и отображение заданий с заголовками и описаниями.
*   **Отправка промптов:** Форма для ввода и отправки пользовательских промптов.
*   **Ответы ИИ:** Отображение сгенерированных ИИ ответов на промпты.
*   **Детализированная обратная связь:** Предоставление оценки промптов по нескольким критериям (Роль и контекст, Чёткость задачи, Полнота данных, Сфокусированность, Структура, Метод решения, Формат ответа), а также общий комментарий и итоговая оценка.
*   **QR-коды:** Генерация QR-кодов для удобного доступа к заданиям с предварительно заполненным ID пользователя.
